# data-viz-app-beta
This is just a quick data visualization application using Streamlit from Python.

<h3>How To Run The Application</h3>
<h4>Install Python 3.15 or Later Before These Steps</h4>
<p>1. Run Command Prompt in Admin Mode</p>
<p>2. Install Streamlit by typing "pip install streamlit"</p>
<p>3. Type "cd 'file path'"</p>
<p>4. Type streamlit run app.py</p>
<p>5. You're done; now the app should run within your browser.</p>
